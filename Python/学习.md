import和from xxx import xxx的区别
例如：import skimage
img = skimage.io.imread(...

from skimage import io
img = io.imread(...
第一种方法只导入了skimage，没有声明io成员。所以在之后的代码中使用io成员都要声明是skimage中的io成员。
第二种方法在导入skimage时就声明是skimage中的io成员，所以在之后使用中不需要再进行声明

30.字符串处理方法
str.lower()或str.upper()  返回字符串的副本，全部字符小写或大写
str.split(sep=None)   返回一个列表，由str根据sep被分隔的部分组成
	如："A,B,C".split(",")  结果为['A','B','C']
str.count(sub)  返回字串sub在str中出现的次数
	如："an apple an day".count("a")  结果为4
str.replace(old,new)  返回字符串副本，所有old字串被替换为new
str.center(width,fillchar)  str根据宽度width居中，fillchar可以省略,但是最多只能填一个字符
	如："python".center(20,"=")  结果为"=======python======="
str.strip(chars)  从str中去掉在其左侧和右侧chars中列出的字符
	如："= python= ".strip(" =p")  结果为"ytho"
str.join(iter)  在iter变量除最后元素外，每个元素后增加一个str,主要用于字符串分隔
	如：",".join("12345")    结果为"1,2,3,4,5"

### PyTorch
torch是PyTorch的核心库，涉及张量、计算等

torch.nn专门用来些神经网络层

	class Encoder(nn.Module):  # 所有PyTorch模型必须继承nn.Module
		def __init__(self, input_dim, hidden_dim):  # input_dim：表示每个时间步有多少个数，hidden_dim：表示模型内部“记忆”的大小
			super().__init__()
			self.lstm = nn.LSTM(input_dim, hidden_dim, batch_first=True)  # input_dim表示每个时间步输入多少数，hidden_dim表示LSTM内部状态大小，batch_first=True表示数据格式是(批次，时间，特征)

		def forward(self, x):  # 模型怎么跑的定义
			_, (h, c) = self.lstm(x)  # LSTM返回output，(h, c)
			# output表示每个时间步的结果，h表示最后一步的记忆，c表示最后一步的长期记忆
			return h, c


	class Decoder(nn.Module):
		def __init__(self, output_dim, hidden_dim, horizon):  # output_dim表示每一步预测几个数，hidden_dim表示模型内部“记忆”的大小，horizon表示要预测多少步未来
			super().__init__()
			self.horizon = horizon
			self.lstm = nn.LSTM(output_dim, hidden_dim, batch_first=True)
			self.fc = nn.Linear(hidden_dim, output_dim)  # 全连接层，把LSTM内部状态转换为真实预测数值

		def forward(self, h, c):  # h和c是Encoder总结的记忆
			batch_size = h.size(1)
			decoder_input = torch.zeros(batch_size, 1, self.fc.out_features).to(h.device)

			outputs = []
			for _ in range(self.horizon):
				out, (h, c) = self.lstm(decoder_input, (h, c))
				pred = self.fc(out)
				outputs.append(pred)
				decoder_input = pred

			return torch.cat(outputs, dim=1)
