7.快速排序
void qsort(void* base, size_t nitems, size_t size, int (\*compar)(const void*, const void*))
其中base为数组首地址，即为数组名本身
size_t为数组元素个数
size为单个元素的字节大小
compar为比较函数，决定函数的排序规则

	int compare(const void* a, const void* b){
		return *(int*)a - *(int*)b;
	}

8.占位符
%d 十进制有符号整数
%u 十进制无符号整数
%f 浮点数
%s 字符串
%c 单个字符
%p 指针的值
%e 指数形式的浮点数
%x 无符号十六进制整数
%o 无符号八进制整数

%3d 表示输出宽位3的整数，不够3位右对齐
%9.2f 表示输出宽为9的浮点数，其中小数位为2，整数位为6，小数点占1位，不够9位右对齐
%04d 表示前面补0
%-4d 表示左对齐
