### 回退commit
git reset HEAD~2 回退两次commit

### 将文件夹传到新建的仓库里
在github上新建一个仓库
git clone 仓库地址
将文件夹里的内容复制到clone的文件中，如果新建的仓库中没有任何内容，会给你报一个warning
git add .
git commit -m "init"
git push

### 新建分支
origin：可以直接在GitHub上创建

本地：git branch 分支名
git checkout 分支名 可以在本地切换到该分支
git push origin 分支名 可以将修改推送到远程仓库

在提交并推送后，git checkout 分支名 可以切换本地分支，本地分支的内容也会在文件夹相应地改变
